{{ 'Approche Configuration du module'|upper }}

Ce chapitre concerne les configurations, nous allons par la suite présenter dans les moindres détails comment utiliser cette partie.

{{ 'Configuration Pension'|upper }}

Il s’agit ici des configurations se rapportant à l’inscription et aux paiements et s’effectue sur l’interface ci-dessous.

Interface d'Accueil sur les Configuration de Pension

Nous allons donc repartir ces configurations par catégorie.

(1) Type Paiement
Interface 'Configuration Pension - Type Paiement'

Le Type de Paiement est un mode de paiement qu’un étudiant peut utiliser pour payer ses frais de scolarité (Espèce, Chèque, Virement Bancaire, etc…).

Comme vous pouvez le voir dans la figure ci-dessus, cette page apparait directement lorsque l’on clique sur le lien « Configuration Pension » du TABLEAU DE BORD SCOLARITE et permet la manipulation des types de paiement : qui nous sera utile lorsque voudra effectuer le paiement des frais de scolarité d’un étudiant dans le sous module PAIEMENT.

Le bouton « Ajouter » (encadrer en jaune) : permet d’ajouter un nouveau type de paiement, le clic sur ce dernier fera apparaitre une mini fenêtre dans laquelle on saisira les informations du nouveau type de paiement (voir figure ci-dessous).

édition d'un Type de paiement

Le clic sur le bouton « Enregistrer » ajoutera le type de paiement dans le tableau en affichant un message de confirmation.

Le lien « Editer » du tableau (encadrer en bleu) : permet d’effectuer des modifications sur le type de paiement choisi. Ce dernier ouvrira la mini fenêtre ci-dessus avec les informations du type de paiement choisi, vous pouvez donc par la suite apporter des modifications comme vous le souhaitez et enregistrer.

(2) Tranche

Il s’agit ici des tranches de paiements des frais de scolarité.

Interface 'Configuration Pension - Tranche'

Cette page permet l’ajout et l’édition des tranches à travers les boutons Ajouter (Jaune) et Modifier (Bleu). Pour pouvoir effectuer une modification sur une tranche faudrait d’abord le sélectionner dans le tableau.

(3) Pension

La Pension est un montant que l’on va attribuer à l’inscription d’un étudiant qui sera considérer comme c’est frais de scolarité.

Il s’agit ici de l’ensemble des pensions effectives dans l’établissement.

Interface 'Configuration Pension - Pension'

Cette page permet l’ajout et l’édition des pensions à travers les boutons Ajouter (Jaune) et Modifier (Bleu). Pour pouvoir effectuer une modification sur une pension faudrait d’abord le sélectionner dans le tableau ci-dessus.

(4) Niveau – Option

Cette partie permet la mise en relation d’une option et d’un niveau.

Interface 'Configuration Pension - Niveau/Option'

Le bouton « Nouveau » (encadré en bleu) permet une nouvelle affectation d’une option à un niveau, le clic sur ce bouton ouvrira une fenêtre dans laquelle on pourra effectuer les configurations comme bon nous semble (voir figure ci-dessous).

Ajout d'un Niveau-Option

Apres avoir sélectionner les différents éléments pour configurer un niveau-option, le bouton « Enregistrer » deviendra actif pour l’enregistrement. S’il ne l’est pas c’est que cette configuration existe déjà, le bouton « Enregistrer » restera donc grisé.

Le lien « Supprimer » (encadré en jaune) permet de supprimer un niveau-option après confirmation s’il n’est pas déjà utilisé.

Le lien « Reconduire » (encadré en rouge) permet de reconduire un niveau-option d’une année à une autre en vue de faciliter la tâche aux utilisateurs.

Reconduction d'un Niveau-Option

La fenêtre ci-dessus apparaitra avoir cliqué sur ce lien, il vous faudra juste choisir l’année académique dans laquelle vous souhaité reconduire le niveau-option et enregistrer. Un message d’erreur apparaitra si le niveau-option sélectionné existe déjà pour l’année en question dans le cas contraire un message de confirmation vous sera affiché.

(5) Profil

Le Profil concerne l’inscription, il définira en quelques sorte le type d’inscription que l’on accordera à l’étudiant. On l’associera à une pension et un niveau-option (voir paragraphe (6) Tranche / Pension).

Interface Configuration - Profil

Cette page liste les différents profils d’inscription.

Le bouton « Ajouter » (encadré en bleu) permet d’ajouter un nouveau profil

Le lien « Editer » (encadré en jaune) permet de faire une édition du profil

NB : Lorsqu’un profil est NON ACTIF il ne pourra pas être utilisé dans le sous module INSCRIPTION ET TRANSFERT.

(6) Tranche / Pension

Cette partie dispose de plusieurs fonctionnalités, dont nous allons détailler leur manipulation ci-dessous. Il s’agit de :

      - Affectation d’une pension à profil et un niveau-option
      - Affectation d’une pension à un profil et plusieurs niveau-options
      - Reconduction d’une affectation d’une année académique à une autre
      - Répartition des tranches en fonction des affectations
Interface Configuration - Tranche/Pension

• Affectation d’une pension à un profil et un niveau-option

Après avoir sélectionner l’année académique ainsi que les différents composants du bloc « Filtre » (encadré en noir), le lien « Affectation Pension/Filière » (encadré en vert) sera actif et un clic dessus ouvrira la fenêtre suivante :

Liaison d'une Pension à un Profil et un Régime pour une option précise

Dans cette fenêtre, on effectue une recherche d’une pension à travers sa pension ensuite on sélectionne la pension retourné dans le tableau et avant d’enregistrer veuillez choisir un profil (voir paragraphe (5) Profil) et un régime (COURS DU JOUR ou COURS DU SOIR).

• Affectation d’une pension à un profil plusieurs niveau-options

Après avoir sélectionné une année académique, le clic sur lien « Affectation Multiple » (encadré en Orange) ouvrira la fenêtre suivante :

Liaison d'une Pension à un Profil et un Régime pour plusieurs options

Dans cette fenêtre, on effectue une recherche d’une pension à travers son montant, on sélectionne la pension retourné dans le tableau. Après avoir choisi l’école, le cycle, le niveau, on choisit les options (encadré en marron) concernées ainsi que le profil et le régime avant d’enregistrer.

NB : Ces affectations nous aidera au niveau de l’inscription d’un étudiant, qui déterminera la scolarité qu’un étudiant devra verser à l’établissement en fonction de sa classe et du profil choisi

• Reconduction d’une affectation d’une année académique à une autre

Dans le but de facilité la taches au utilisateur, on peut reconduire une affectation d’une année antérieur à celle que l’on a sélectionné. Pour cela, il suffit de cliquer sur le lien « Reconduire » (encadré en bleu) et la fenêtre suivante :

fenêtre de reconduction des affectations pour une année académique

Cette fenêtre liste les différentes affectations des années antérieures à celle choisi et possède un filtre par école et cycle. Il suffit donc de cliquer sur le lien « Reconduire » (encadré en bleu) d’une affection pour effectuer une reconduction après confirmation. Si l’affectation existe déjà pour l’année en cours un message d’erreur vous sera renvoyer dans le cas contraire ce sera plutôt un message de confirmation.

• Répartition des tranches en fonction des affectations

Il s’agit ici de repartir une affectation en plusieurs tranches (plus connu comme la répartition de la pension en tranche). Et pour le faire il suffit de cliquer sur le lien « Tranches Affectées » (encadré en rouge), s’ouvrira alors la fenêtre suivante :

Fenêtre de création des tranches d'une pension

Cette fenêtre de lister les différentes repartions de tranches pour une affectation. On peut ainsi Ajouter, Editer ou supprimer une tranche attribuée.

Le clic sur le bouton « Ajouter » ouvre la fenêtre suivante :

Fenêtre d'ajout d'une tranche

Qui permettra de configurer l’attribution d’une tranche.

La même interface mais au préalablement remplit s’ouvrira lorsqu’on clique sur le lien « Editer ».

La suppression s’effectue lorsqu’on clique sur le lien « Supprimer ».

(7)Tarifs Niveau-Option

Cette partie a pour principal objectif l’attribution des montants à des niveaux-options (voir paragraphe niveau-option) qui nous sera utile dans le module INSCRIPTION ET TRANSFERT à travers l’interface suivante :

Interfaces Configuration - Tarifs Niveau/Option
L’espace filtre

L’espace filtre (encadré en Noir) permet de filtre les niveaux-options en fonction de l’école, le cycle, la filière, la spécialité, l’option et le niveau sur l’année académique choisi.

Le lien « Editer »

Le clic sur le lien « Editer » (encadré en jaune) sur un élément du tableau permettra d’ouvrir l’interface suivante :

Edition d'un Niveau-Option (sur son prix)

Rempli au préalable ou non en fonction du niveau-option choisi. Si le niveau-option possédait déjà des informations il s’ouvrira déjà rempli sinon il s’ouvrira vide. Dans les deux cas on saisit les informations et on enregistre.
