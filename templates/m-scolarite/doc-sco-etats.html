Dans ce chapitre nous verrons comment manipuler les états du module.

{{ 'ETATS SUR LES CLASSES'|upper }}

Cette partie est spécialisée sur les états donnants des informations sur les classes ou sur ses étudiants. L’interface suivante est celle dans laquelle ces traitements s’effectuent.

Interface de gestion des états de classe

Nous allons par la suite expliquer chaque composant de l’interface.

(1) Année Académique

Le choix de l’année académique permet de déterminer dans quelle année on va travailler. C’est le point de départ des manipulations.

(2) Le lien « Impression des effectifs »

Ce lien permet d’ouvrir la fenêtre donnant accès au choix d’impression.

Fenêtre de choix d'impression

Dans laquelle on choisit le type d’impression que l’on veut faire avant d’imprimer.

NB : Ici les impressions ne sont portées que sur les effectifs des classes.

(3) Le choix « Impression Groupée Par Classe »

Ce choix permet de faire des impressions pour plusieurs classes en même temps en activant et désactivant le lien « Impression Multiple » (5).

(4) Le Filtre « Ecole / Cycle / Classe »

Ce filtre permet le choix de la classe permettant ainsi de lister dans le tableau (8) les étudiants inscrits dans la classe choisi.

(5) Le lien « Impression Multiple »

Le clic sur ce lien va ouvrir la fenêtre suivante.

Impression groupée par classe

Dans laquelle on a plus qu’à sélectionner les classes et choisir le type d’impression que l’on souhaite avant d’imprimer. L’impression peut se faire en PDF ou sur un fichier Excel.

NB : Les types d’impression concerne la liste des étudiants de la classe selon différents critères.

(6) Bloc « Impression »
Bloc d'impression

Après avoir choisi une classe (4) ce bloc va imprimer une liste d’étudiant de la classe selon le critère d’impression, avant d’imprimer en PDF ou en Excel.

(7) Choix d’Impression
Impressions particulières

Ce menu s’active après avoir choisi l’année académique et permet une impression directe en fonction du menu choisi.

(8) Tableau d’Etudiant
Liste des étudiants de la classe

Ce tableau liste les étudiants de la classe choisi (4).

Le Lien « Certificat de scolarité »

Le clic sur ce lien permet d’imprimer le certificat de scolarité de l’étudiant choisi en ouvrant la fenêtre suivante :

Choix de la langue avant impression d'un Certificat

Dans laquelle on choisit en quelle langue on veut le certificat de scolarité avant d’imprimer.

NB : Cette action affichera une erreur si on n’a pas configuré les tranches pour l’option de la classe choisi (voir paragraphe APPROCHE CONFIGURATION DU MODULE / (6). Tranche / Pension) ou si l’étudiant n’as pas encore fini le versement de la première tranche.

Le lien « Détails sur l’inscription »

Le clic sur ce lien permet d’imprimer un tableau présentant les soldes de l’étudiant dans la classe par tranche.
